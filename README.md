# Next.js Project Setup and Basics

This repository is part of the **ALX curriculum** and is focused on setting up and understanding the basics of a **Next.js project**.  

### Purpose
The goal of this repository is to:
- Learn how to properly initialize a Next.js project.
- Understand the project structure and configurations.
- Explore basic features such as pages, routing, and components.
- Build a strong foundation for advanced projects using Next.js.
